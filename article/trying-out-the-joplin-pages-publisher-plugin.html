First blog article

What’s a good topic for the first article in a blog? In my case, the technical system used for setting up this blog feels like a natural choice. Starting a blog, like many projects, comes with an initial inertia one needs to overcome, especially if you do not intend on using some ready-made 3rd party offer.

As I am already using Joplin to manage my todos and organize my thoughts on whatever topic I am dealing with, this plugin hits a lot of points that are attrative to me in a blog management system:

  • uses markdown to write the actual articles
  • uses a git repository for version control
  • comes with a simple default template…
  • …which can easily be extended (just some EJS templates)
  • simple deployment workflow

Which together with me already using Joplin reduces the effort to get started to about five minutes of configuration and then, naturally, the time needed to write an article. Fairly ideal for my situation.

Outlook

I will have to see if this is enough to keep me using this or if it is just a one-off attempt. If I keep using this, I will probably play around and customize the default template a bit. If this really grabs my attention, I might actually extend the plugin to add support to use sourcehut as a backend, which I would prefer over github in the long term.

This should be enough text for a first article.
